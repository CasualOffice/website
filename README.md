# Casual Office вҖ” schnsrw.live

The home of **Casual Office** вҖ” an open-source, file-centric, real-time
productivity suite by [Sachin Sarwa](https://github.com/schnsrw). Three
products live under the umbrella:

- **Casual Sheets** вҖ” Excel-flavored web spreadsheet В· [demo](https://sheet.schnsrw.live/) В· [repo](https://github.com/schnsrw/sheets)
- **Casual Editor** вҖ” Word-flavored web `.docx` editor В· [demo](https://doc.schnsrw.live/) В· [repo](https://github.com/schnsrw/docx)
- **Casual Desktop** вҖ” Tauri binaries, in progress

Built with **Astro 5** (static output) and deployed to GitHub Pages on
every push to `main`. The site is **`schnsrw.live`** via the custom
domain in `public/CNAME`.

## Deploy

`main` push вҶ’ `.github/workflows/pages.yml` runs:

1. Checkout
2. `npm install`
3. `npm run build` (Astro emits to `./dist`)
4. Upload `./dist` as a Pages artifact
5. `actions/deploy-pages@v4`

CNAME (`public/CNAME` вҶ’ ships as `dist/CNAME`) tells Pages to serve
on **`schnsrw.live`**. DNS apex points to GitHub's IPs:

## SEO posture

- **Meta** вҖ” every page gets full Open Graph, Twitter card, canonical,
  JSON-LD `SoftwareApplication` / `Person` / `WebSite`. Lives in
  `src/layouts/Base.astro` + per-page `jsonLd` prop.
- **Sitemap** вҖ” auto-generated by `@astrojs/sitemap` (see
  `astro.config.mjs`); the home + per-product pages get bumped priority.
- **robots.txt** вҖ” `public/robots.txt` opts in every major AI crawler
  by name (GPTBot, ClaudeBot, PerplexityBot, OAI-SearchBot, CCBot,
  Applebot-Extended, Google-Extended, вҖҰ) and the social-unfurlers.
- **llms.txt** вҖ” `public/llms.txt` carries the long-form LLM-friendly
  description (emerging convention).
- **OG image** вҖ” `public/og.png` (1200 Г— 630). Regenerate with
  `node scripts/build-og.mjs` from a directory that has Playwright
  installed (typically `../sheet/`).
